# Portfolio Website

## Overview
This is my personal portfolio website built with **Next.js**, **Tailwind CSS**, and **React**.  
It showcases my **projects, experience, and contact information**, with **dark mode support** and **responsive design** for desktop and mobile.

**Website Link:** https://personal-website-three-sigma-94.vercel.app/

### Features
- **Responsive Design**: Works on desktop, tablet, and mobile devices  
- **Dark Mode Toggle**: Switch between light and dark themes  
- **Animated Components**: Motion effects on sections and buttons using `motion/react`  
- **Project Showcase**: Dynamic project cards with hover effects  
- **Contact Form**: Integrated with [Web3Forms](https://web3forms.com/) for form submission  

---

## Tech Stack
- **Framework**: Next.js  
- **Styling**: Tailwind CSS  
- **Animations**: Framer Motion (via `motion/react`)  
- **Deployment**: Vercel 
- **Other Tools**: GitHub for code hosting, Web3Forms for contact form

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)  
- npm or yarn  

### Installation
```bash
# Clone the repo
git clone https://github.com/andynyugen/portfolio-website.git

# Navigate into the project
cd portfolio-website

# Install dependencies
npm install
# or
yarn install

# Run locally
npm run dev
# or
yarn dev
